171 # example showing how to use sh
172 # it sends a command to the fifo, whose path is told via a positional param
173 # if fifo_dir is not set, it'll echo to a file named (null) somewhere >:) remember to delete it
174 # The body of the shell command should be one parameter, so if it has spaces like here,
175 # you must enclose it in quotes.  Remember to escape (and double-escape) quotes and backslashes
176 # in the body.  Any additional parameters you use will appear AFTER the default parameters (cfg file
177 # path, fifo & socket dirs, etc.)
178 @bind    XS        = sh 'echo "js alert (\\"This is sent by the shell via a fifo\\")" > "$4"'
